    # 场景操作

    案例名称 / 链接 案例内容 图例

    Camera (opens new window) 相机的各种移动方法示例

    Cesium Inspector (opens new window) Cesium监视器，用于查看相机视锥体、调试图形Primitives、调试地形

    Globe Interior (opens new window) 开启地表影像半透明，用于查看地球内部

    Globe Translucency (opens new window) 修改地表透明度
    HTML Overlays (opens new window) 附着在地球上的HTML元素
    LocalToFixedFrame (opens new window) 不同坐标系下的变换
    Multiple Synced Views (opens new window) 二维/三维场景联动效果
    Parallels and Meridians (opens new window) 显示经线和纬线
    Picking (opens new window) 坐标拾取、实体Entity拾取
    Projection (opens new window) 相机投影方式：orthographic正交投影、perspective透视投影（默认投影方式）
    Resolution Scaling (opens new window) 分辨率缩放比率：可以使用浏览器默认的比率，也可以自定义比率，比率越小场景越模糊（马赛克效果），反之越清晰。比率越大，浏览器渲染压力也就越大
    Rotatable 2D Map (opens new window) 旋转2D场景下的地图
    Scene Rendering Performance (opens new window) 场景性能测试：requestRenderMode请求渲染模式
    Underground Color (opens new window) 修改地表颜色
    Video (opens new window) 在地球表面播放视频
